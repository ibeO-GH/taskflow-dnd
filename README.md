# TaskFlow – Drag & Drop Task Manager

TaskFlow is a modern task management application inspired by tools like Trello. It allows users to manage tasks across different stages using an intuitive drag-and-drop interface.

## 🚀 Features

- Drag and drop tasks between columns
- Task status management (Pending → In Progress → Completed)
- Inline editing of tasks
- Dropdown actions (Start, Complete, Delete)
- Clean and responsive UI
- Smooth animations and interactions

## 🛠️ Tech Stack

- React
- TypeScript
- Tailwind CSS
- dnd-kit

## 📦 Installation

```bash
npm install
npm run dev
```

## 🏗️ Build

```bash
npm run build
```
